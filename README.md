# word_card
## NTOU 111學年度網頁程式設計第14組期末專案

## Requirements
+ Python 3.8.13+

## Build Steps

### Install environment and dependencies
```
python -m venv venv
.venv/bin/activate
pip install -r requirements.txt
```

### Run it

```
python app.py
```

### View it on web browser and play fun

```
http://localhost:8080
```
